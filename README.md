# Python Port Scanner

## Overview
This project is a basic TCP port scanner built using Python's socket library. It scans a target IP address across a specified port range and identifies open ports.

## Features
- Custom IP input
- Custom port range
- Timeout handling
- Clear output of open ports

## Technologies Used
- Python
- socket library

## How to Run
1. Clone the repository
2. Run: python port_scanner.py
3. Enter target IP and port range when prompted

## Disclaimer
This tool is for educational purposes only. Only scan systems you own or have permission to test.
